# 0.1.0 — 2026-06-25

### 功能

- 仪表盘：体力环、2×3 状态格（签到、派遣、周本、训练）
- 挑战：混沌回忆、虚构叙事、末日幻影、挑战峰值 + 周进度条
- 活动·档案：星琼月报、卡池信息、限时活动倒计时、模拟宇宙常驻存档
- 系统托盘：常驻通知区图标、右键菜单
- 深色/浅色主题切换
- 桌面通知：体力满提醒、派遣完成等
- 通知模式：纯系统通知模式（无 UI 窗口）
- Onboarding 引导：首次启动向导、数据目录自定义、WebView 登录
- 设置页面重构：iOS 风格分组列表 + 下钻导航 + 自动保存
- 通知规则配置：体力阈值滑块、签到/模拟宇宙提醒时间、每日摘要
